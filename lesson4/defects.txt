Дефект #1
Название: TypeError при вызове метода capitilize с целочисленным аргументом
Описание: Метод capitilize вызывает ошибку TypeError, когда ему передается аргумент типа int вместо строки.
Шаги к воспроизведению:
1. Создать экземпляр класса StringUtils.
2. Вызвать метод capitilize, передав в качестве аргумента целое число (например, 123).
Ожидание: Метод должен работать только со строками и выдавать ошибку, если передан другой тип данных.
Реальность: Возникновение ошибки TypeError.
Критичность: Средняя
Дата обнаружения: 01.10.2023
Исправлено: Нет
Комментарии: Необходимо добавить проверку типа входного параметра и обрабатывать исключение.

Дефект #4

2. Вызвать метод list_to_string, передав в качестве первого аргумента список элементов, а в качестве второго аргумента None.
Ожидание: Метод должен генерировать строку, используя заданный разделитель между элементами списка.
Реальность: Возникновение ошибки TypeError.
Критичность: Высокая
Дата обнаружения: 04.10.2023
Исправлено: Нет
Комментарии: Необходимо предусмотреть корректную работу метода при передаче значения None в качестве разделителя.
